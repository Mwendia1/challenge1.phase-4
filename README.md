# Superheroes Flask API

## Date: 2025/12/03
## By: Elijah Mwendia

### Description

- The Superheroes Flask API is a backend web application built using Flask that manages superheroes, their powers, and the relationships between them.
- The application replaces static or manual superhero data management with a structured RESTful API that supports creating, reading, and updating heroes and powers while enforcing validations and relationships.

- The API follows REST principles and returns JSON responses that can be tested using the provided Postman collection.

### Features

- View a list of all heroes

- View a single hero along with their powers

- View all powers

- View a single power

- Update an existing power’s description


### Learning Goals

- Understand how to build relational data models using SQLAlchemy

- Practice implementing many-to-many relationships

- Learn how to apply model validations

- Build RESTful routes using Flask

- Handle errors and HTTP status codes correctly

- Use Postman to test API endpoints
